Backend Platform – Modular Monolith (Multi-Domain)

A backend platform built using a modular monolith architecture with multiple business domains including: Pharmacy, Real Estate, Automobile marketplace, Notifications and shared services such as authentication and payments.

Business Features

  • JWT Authentication + Role access
  • Real Estate Listings + Checkout
  • Automobile Marketplace Modules
  • Pharmacy Inventory Management
  • Stripe + Flutterwave Payments
  • GPS Tracking Service
  • Email / SMS Notification Service
  • Admin Back Office Panels

Architecture Summary

  • Domain Driven Design (bounded contexts)
  • REST APIs per domain
  • PostgreSQL + Prisma ORM
  • Node.js + TypeScript backend services
  • Scalable evolution to microservices if required
